# Documentación: prompts.md

## Propósito

Registra los prompts más relevantes utilizados durante la creación del proyecto MVP, organizados por sección funcional. Sirve como trazabilidad del uso de IA y referencia para prompts efectivos.

## Ubicación

`prompts.md` (raíz del proyecto)

## Estructura

Cada entrada de prompt sigue el formato:
- **Prompt utilizado:** Texto exacto del prompt
- **Herramienta:** ChatGPT, Cursor IDE, etc.
- **Resultado:** Artefactos generados o cambios realizados
- **Ajustes humanos:** (opcional) Correcciones o validaciones realizadas
- **Referencia:** (opcional) Archivos relacionados

## Secciones

- Producto
- Arquitectura
- Modelo de Datos
- Historias de Usuario
- API / Endpoints
- Reglas de Negocio y Validaciones
- Backend - Modelos
- Frontend
- Especificaciones de Modelos Backend
- Manual del Programador (Prompt 13)
- Notas sobre el Uso de IA
- Formato de Fechas
- Generación Masiva de TRs
