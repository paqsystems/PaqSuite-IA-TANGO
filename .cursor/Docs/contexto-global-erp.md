# Documentación: contexto-global-erp.md

## Ubicación
`docs/00-contexto/contexto-global-erp.md`

## Propósito
Este documento define el marco conceptual general de la plataforma ERP corporativa: modelo multiempresa, multiusuario y multirrol, base central de configuración, bases operativas por empresa, modelo de seguridad y principios rectores del sistema.

## Contenido Principal

- Propósito del proyecto y modelo conceptual general
- Base de datos "Diccionario" (contexto central)
- Bases de datos por empresa (contexto operativo)
- Modelo de seguridad y resolución de tenant
- Principios rectores y alcance macro del ERP

## Uso
Consultar este documento para comprender el contexto global antes de cualquier tarea técnica relacionada con el ERP.
