# Control de Calidad de Usuarios

Registro de errores detectados en **pruebas manuales** realizadas por usuarios. Se conserva el historial de cada control con fecha, autor y estado. El estado se inicializa en **Pendiente** hasta que se generen las HU correspondientes y se resuelvan los errores.

> **Nota:** Previsto: regla en `.cursor/rules/` sobre cómo generar las HU a partir de los errores y cómo actualizar el estado de cada control.
